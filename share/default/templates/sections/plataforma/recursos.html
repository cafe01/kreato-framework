Painel de Controle Intuitivo

Nosso Painel de Controle foi projetado para ser utilizado apenas pelos administradores do website, diferente de outras soluções que misturam funções de desenvolvedor na interface. Isso nos permitiu desenvolver uma interface simples e extremamente intuitiva.

Componentes

Websites são compostos por páginas que são compostas por componentes, como portfolios, galerias, agendas, blogs, entre muitos outros. A organização do conteúdo dinâmico dessa forma é o que possibilita total liberdade para criação de qualquer tipo de website. Sem gambiarras.

Componentes

Websites são compostos por páginas que são compostas por componentes, como portfolios, galerias, agendas, blogs, entre muitos outros. A organização do conteúdo dinâmico dessa forma é o que possibilita total liberdade para criação de qualquer tipo de website. Sem gambiarras.

Redes Sociais

Além de possuir componentes com funções específicas, como álbum do Instagram, lista de vídeos do Youtube, entre outros, a Plataforma Kreato já está preparada para se comunicar com as APIs das principais redes sociais. Ou seja, podemos desenvolver soluções exclusivas para os seus projetos. Já imaginou uma ação promocional envolvendo hashtags do Instagram ou seções do website que apareçam apenas para fans de uma Fan Page do Facebook? A criatividade é o limite!

Banco de Dados Flexível

Projetado com a metodologia EAV (Entity Attribute Value), nosso sistema permite modelagens de dados diferentes para cada projeto. Isso nos permite atender necessidades específicas de processamento de dados.

Banco de Dados Flexível

Projetado com a metodologia EAV (Entity Attribute Value), nosso sistema permite modelagens de dados diferentes para cada projeto. Isso nos permite atender necessidades específicas de processamento de dados.

Sistema de Pagamento Online

Apesar de ainda não possuírmos uma solução completa de E-Commerce, nossa plataforma já é integrada com serviços de pagamento online, como PagSeguro, Paypal e MOIP. Você pode, por exemplo, gerenciar as mensalidades de um serviço online ou offline, criar uma área restrita do website apenas para assinantes ou até mesmo criar um pequeno comércio online.

Sistema de Template

Diferentes de outras plataformas onde os arquivos de templates são uma mistura de html e códigos de programação, no nosso sistema de template, esses arquivos são compostos por html puro que invocam os componentes através de classes css. Isso agiliza o desenvolvimento e elimina uma série de possíveis erros de programação, inclusive falhas de segurança.

White Label

Oferecemos a opção White Label, na qual substituímos as logomarcas da nossa plataforma pela da sua agência ou cliente (ex: logo na tela de login, logo do Painel de Controle, etc) agregando ainda mais valor à sua venda e claro à sua empresa.

Sistema de Estatísticas

Seus clientes poderão monitorar o número de visitas ao website em um determinado período, quais navegadores e dispositivos estão sendo usados para o acesso, de quais regiões são os visitantes e muito mais, tudo pelo próprio Painel de Controle do site. Além disso, este sistema também pode ser customizado para gerar estatísticas e gráficos para dados específicos de um projeto.

Com o Kreato Drive seus clientes contarão com uma solução completa para compartilhar pastas e arquivos em nuvem, assim como gerenciar os usuários e suas devidas permissões de acesso. Tudo rápido e fácil em uma interface bem intuitiva.
