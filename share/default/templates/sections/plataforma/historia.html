Buscando fornecer o melhor serviço de desenvolvimento web possível, avaliamos as plataformas disponíveis no mercado para criação de sites e aplicativos web, como Wordpress, Joomla, Drupal, entre outros. Infelizmente nenhuma delas atendiam aos seguintes critérios:

Multi Tenancy

Ser capaz de oferecer um serviço web para atender a múltiplos clientes. Plataformas que não atendem este modelo precisam ser instaladas em ambientes diferentes para cada cliente, ou seja, cada site é uma nova hospedagem a ser gerenciada com todos custos e tarefas atribuídas. Fica claro que a medida que cresce o número de clientes, torna-se inviável o gerenciamento de tantas hospedagens diferentes. Saiba mais

Banco de Dados Flexível

Oferecer uma maneira simples e flexível para criar novas modelagens de dados que possam atender necessidades específicas de um projeto. Saiba mais

Template Seguro

Códigos de programação inseridos nos templates não podem ter acesso a recursos que botem os sites ou a plataforma em risco.

Plugin Free

Deve oferecer todos recursos necessários para criação de websites sem a necessidade da instalação de plugins de terceiros, principal fonte de vulnerabilidade e incompatibilidade.

Imparcial

Não deve ser construída para tipos específicos de websites, como Blogs, E-Commerce, entre outros. Tais funcionalidades devem ser fornecidas através de módulos.

White Label
